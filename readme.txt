Enhance your WooCommerce store with instant SMS alerts for order updates and engage customers with automated messages using unify.smsgateway.center SMS API.

== Description ==

Elevate your WooCommerce experience with unify.smsgateway.center SMS notifications, a powerful solution for sending automated SMS notifications for order statuses. unify.smsgateway.center, a global leader in bulk SMS messaging services, provides seamless integration with WooCommerce, enabling both administrators and customers to receive real-time updates directly through SMS. Customize alert texts, manage Delivery Report Logs (DLR), and engage customers with bulk messaging to subscriber groups - all from your WordPress dashboard.

**Key Features**:
- National and International SMS Messaging via unify.smsgateway.center SMS API.
- Automated alerts for WooCommerce order events.
- Customizable SMS messages for each order status.
- Two Way auto-response for unlimited keywords via unify.smsgateway.center.
- View and manage DLR logs directly in WooCommerce.
- Bulk SMS messaging to defined subscriber groups.
- Compatible with WooCommerce Version 8.2.2 and above.
- Integration of Operators approved DLT templates.

**Supported Events**:
- Order statuses: Completed, Processing, Pending Payment, On Hold, Cancelled, Refunded, Failed.
- Customer engagements: New Registration Welcome, Password Reset, Product Review Submission/Approval, Profile Updates.
- Admin notifications for all customer events.

== Installation ==

= WordPress Dashboard =
1. Download the plugin zip file.
2. Log in to your WordPress admin dashboard.
3. Navigate to *Plugins > Add New*.
4. Click *Upload Plugin* and select the downloaded zip file.
5. Click *Install Now* and *Activate Plugin* after installation.
6. Find the new "Unify SGC SMS" menu in the sidebar.

= FTP Method =

1. Unzip the downloaded plugin folder.
2. Connect to your server using FTP.
3. Upload the plugin folder to wp-content/plugins/.
4. Log in to your WordPress admin dashboard.
5. Navigate to *Plugins* and activate the Woo SMS Connect Unify SMS Gateway Center plugin.
6. A new "Unify SGC SMS" menu will appear in the sidebar.
